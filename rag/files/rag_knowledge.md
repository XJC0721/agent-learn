# 大模型应用开发知识库

## 一、RAG 核心概念

RAG（Retrieval Augmented Generation，检索增强生成）是一种将外部知识库与大语言模型结合的技术框架。其核心思路是：在模型生成答案之前，先从知识库中检索与问题相关的文本片段，再将这些片段作为上下文一起传给模型，从而让模型"有据可查"地回答问题。

RAG 解决了大语言模型的三个主要问题：
- **知识过期**：模型训练数据有截止日期，RAG 可以接入实时更新的知识库
- **幻觉问题**：模型缺乏知识时会编造答案，RAG 提供了可信的参考来源
- **私有数据**：企业内部文档、产品手册等私有数据无法通过训练获得，RAG 可以直接接入

## 二、RAG 的四个步骤

**第一步：数据准备**
将原始文档（PDF、Markdown、Word 等格式）加载进来，按照一定策略切分成小的文本块（chunk）。常用切分方式包括：按固定字符数切分、按段落切分、按语义相似度切分。

**第二步：建立索引**
对每个文本块调用 Embedding 模型进行向量化，将文字转换为一串浮点数（向量），存入向量数据库。向量数据库会为每条数据同时保存原文、向量和元数据。

**第三步：检索**
用户提问时，将问题同样向量化，然后在向量数据库中做相似度搜索，找出语义上最接近的 k 个文本块。常用相似度算法有余弦相似度和欧氏距离。

**第四步：生成答案**
将检索到的文本块拼接成上下文，连同用户问题一起传入 LLM，通过 Prompt 引导模型基于上下文生成答案。

## 三、文本切片策略

文本切片是 RAG 效果的关键环节，常见策略如下：

**固定大小切片**：按照固定字符数（如 500 字）切分，简单高效，但可能切断句子导致语义不完整。通常配合 overlap（重叠）来缓解这个问题，例如 chunk_size=500、chunk_overlap=50。

**递归字符切分（RecursiveCharacterTextSplitter）**：LangChain 提供的默认切分器，按照段落、句子、字符的优先级递归切分，在保证块大小的同时尽量保持语义完整。

**语义切分**：基于句子嵌入的相似度来决定切分点，相似度低的地方才切分，语义连贯性最好，但计算成本较高。

## 四、检索优化技术

基础向量检索存在一些局限，常用的优化技术包括：

**混合检索（Hybrid Search）**：同时使用向量检索（语义相似）和关键词检索（BM25），两者结果融合后排序，兼顾语义理解和精确匹配。

**查询改写（Query Rewriting）**：用 LLM 将用户的原始问题改写成更适合检索的形式，或扩展成多个角度的子问题，提高召回率。

**重排序（Reranking）**：向量检索召回粗排结果后，再用一个专门的重排模型（如 BGE-Reranker）对结果精排，提升最终 top-k 的质量。

**HyDE（假设文档嵌入）**：先让 LLM 根据问题生成一段假设性答案，再用这段答案去检索，比直接用问题检索效果更好，因为答案和文档的语义空间更接近。

**父文档检索（Parent Document Retrieval）**：用小块文本做检索（粒度细，精度高），但返回该小块所属的大块文本给 LLM（上下文完整），兼顾精度和完整性。

## 五、向量数据库对比

| 数据库 | 特点 | 适用场景 |
|--------|------|----------|
| Faiss | Facebook 出品，纯本地库，速度极快 | 中小公司、单机部署 |
| Milvus | 分布式，支持百亿级数据 | 大厂、高并发场景 |
| Chroma | 轻量，开发调试友好 | 原型开发、学习使用 |
| Qdrant | 支持稠密+稀疏混合检索 | 需要混合检索的场景 |
| ElasticSearch | 老牌搜索引擎，支持向量检索 | 已有 ES 基础设施的团队 |

向量数据库的核心技术是近似最近邻搜索（ANN），常用算法为 HNSW（层级可导航小世界图），它通过构建多层图结构实现在大规模数据中的快速近似检索，精度和速度之间可以通过参数调节平衡。

## 六、RAG 评估方法

RAG 系统的评估分为两个维度：

**检索质量**：
- 召回率（Recall）：相关文档有多少被检索到了
- 精确率（Precision）：检索到的文档有多少是真正相关的

**生成质量**：
- 忠实度（Faithfulness）：答案是否完全基于检索内容，没有编造
- 答案相关性（Answer Relevancy）：答案是否真正回答了用户的问题
- 上下文相关性（Context Relevancy）：检索到的上下文是否与问题相关

常用评估框架包括 RAGAS 和 TruLens，它们可以自动化地对上述指标打分。
